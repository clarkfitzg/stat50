Let $X$ be a a continuous R.V. with Probability Density Function (PDF) $f(x)$.

\begin{itemize}

\item
The mean of $X$ is:
\[
\mu_X = \int_{-\infty}^{\infty} x f(x) dx
\]

\item
The Cumulative Distribution Function (CDF) for $X$ is:
\[
F(x) = P(X \leq x) = \int_{-\infty}^x f(t) dt
\]

\end{itemize}

\begin{questions}

\question
Answer these in the form $a \leq x \leq b$, for appropriate $a, b$, possibly $\infty$.

\begin{parts}
\part
What are the possible values for a random variable $X$?


\part
What are the possible values for a PDF $f(x)$?


\part
What are the possible values for a CDF $F(x)$?


\end{parts}


\question

\begin{parts}

\end{parts}
